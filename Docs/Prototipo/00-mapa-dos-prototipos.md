# 00 — Mapa dos Protótipos do PopFlow

Este arquivo organiza os protótipos HTML disponíveis em `/Docs/Prototipo`.

> Regra: os protótipos são referência visual/UX. Regras funcionais e de negócio devem seguir os PRDs em `/Docs/PRD`.

## Mapa por tela

- `01-dashboard-popflow.html` → Dashboard geral
- `02-listagem-de-pops-popflow.html` → Listagem de POPs
- `03-detalhe-do-pop-popflow.html` → Detalhe do POP
- `04-criar-pop-popflow.html` → Criação/Edição de POP
- `05-execucao-guiada-midia-contextual-popflow.html` → Execução guiada com mídia contextual
- `06-revisoes-popflow.html` → Revisões e aprovações
- `07-historico-de-versoes-popflow.html` → Histórico e versionamento
- `08-biblioteca-de-templates-popflow.html` → Biblioteca de templates
- `09-comentarios-e-colaboracao-popflow.html` → Comentários e colaboração
- `10-analytics-popflow.html` → Analytics operacional
- `11-notificacoes-popflow.html` → Notificações
- `12-usuarios-e-permissoes-popflow.html` → Usuários e permissões
